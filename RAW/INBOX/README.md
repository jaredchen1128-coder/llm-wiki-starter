# INBOX

Languages: [English](README.md) | [中文](README.zh-CN.md)

This is the temporary capture inbox for new material.

When you find an article, blog post, web clipping, screenshot note, transcript, course material, or any item that is not yet classified, put it here first.

## Rules

- Capture first. Do not worry about perfect classification.
- During ingestion, the Agent decides whether the material is worth adding to `WIKI`.
- If the material has long-term value and its category is clear, the Agent may move it to the most relevant `RAW` category folder.
- If the category is unclear but the material is valuable, it can stay in `RAW/INBOX` and be marked as `review-needed` in `WIKI/ingest-register.md`.
- If the material has no long-term value, it can be marked as `skipped`.

## Recommended Prompt

```text
I added new material. Please scan and ingest it into the knowledge base.
```

## Expected Result

After ingestion, the Agent should update:

- `WIKI/sources/`
- related topic, concept, question, or decision pages
- `WIKI/ingest-register.md`
- `WIKI/index.md`
- `WIKI/log.md`
