# INBOX

语言：[English](README.md) | [中文](README.zh-CN.md)

这里是新资料的临时入口。

看到文章、博客、网页剪藏、截图说明、临时笔记、课程材料或其他暂时不知道如何分类的内容时，先放到这里。

## 使用规则

- 先收集，不急着分类。
- 整理时由 Agent 判断资料是否值得进入 `WIKI`。
- 如果资料值得长期保存，且分类明确，Agent 可以将它移动到最合适的 `RAW` 分类文件夹。
- 如果暂时无法判断分类，可以继续留在 `RAW/INBOX`，并在 `WIKI/ingest-register.md` 标记为 `review-needed`。
- 如果资料没有长期价值，可以标记为 `skipped`。

## 推荐触发语

```text
我新增了资料，帮我扫描并入库
```

## 整理结果

整理完成后，Agent 应更新：

- `WIKI/sources/`
- 相关主题页、概念页、问题页或决策页
- `WIKI/ingest-register.md`
- `WIKI/index.md`
- `WIKI/log.md`
